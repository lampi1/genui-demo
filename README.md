# genui-cv

A single-page demo that explains **generative UI** by *being* generative UI: instead of
a static article, visitors chat — and every answer is an interface (cards, diagrams,
comparisons, timelines) generated by the model on the spot.

Two generation modes, one design system:

1. **Tool-calling components** — curated React components the model invokes with typed props.
2. **Declarative UI spec** — an A2UI-inspired JSON payload rendered client-side for open-ended answers.

Runs for **0 EUR/month**: Vercel Hobby hosting + Gemini Flash free tier.

## Quickstart

```bash
npm install
npm run dev     # http://localhost:3000
npm run test    # Vitest
npm run lint    # ESLint
```
